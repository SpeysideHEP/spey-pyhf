Citation
========

The BibTeX entry for citing ``spey`` is

.. code-block:: BibTeX

    @article{Araz:2023bwx,
        author = "Araz, Jack Y.",
        title = "{Spey: smooth inference for reinterpretation studies}",
        eprint = "2307.06996",
        archivePrefix = "arXiv",
        primaryClass = "hep-ph",
        reportNumber = "IPPP/23/34",
        month = "7",
        year = "2023"
    }

    @software{spey_pyhf_zenodo,
        author    = {Jack Y. Araz},
        title     = {SpeysideHEP/spey-pyhf: v0.1.4},
        month     = jan,
        year      = 2024,
        publisher = {Zenodo},
        version   = {v0.1.4},
        doi       = {10.5281/zenodo.10569195},
        url       = {https://doi.org/10.5281/zenodo.10569195}
    }

.. code-block:: BibTeX
    
    @software{pyhf,
        author = {Lukas Heinrich and Matthew Feickert and Giordon Stark},
        date-added = {2023-05-23 12:36:09 pm +0100},
        date-modified = {2023-05-23 12:36:09 pm +0100},
        doi = {10.5281/zenodo.1169739},

        title = {{pyhf: v0.7.2}},

        version = {0.7.2},
        bdsk-url-1 = {https://doi.org/10.5281/zenodo.1169739}}
