Citation
========

The BibTeX entry for citing ``spey`` is

.. code-block:: bibtex

    @article{Araz:2023bwx,
        author        = {Araz, Jack Y.},
        title         = {{Spey: Smooth inference for reinterpretation studies}},
        eprint        = {2307.06996},
        archiveprefix = {arXiv},
        primaryclass  = {hep-ph},
        reportnumber  = {IPPP/23/34},
        doi           = {10.21468/SciPostPhys.16.1.032},
        journal       = {SciPost Phys.},
        volume        = {16},
        number        = {1},
        pages         = {032},
        year          = {2024}
    }
