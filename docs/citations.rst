Citation
========

The BibTeX entry for citing ``spey`` is

.. code-block:: bibtex

    @article{Araz:2023bwx,
        author = "Araz, Jack Y.",
        title = "{Spey: smooth inference for reinterpretation studies}",
        eprint = "2307.06996",
        archivePrefix = "arXiv",
        primaryClass = "hep-ph",
        reportNumber = "IPPP/23/34",
        month = "7",
        year = "2023"
    }
